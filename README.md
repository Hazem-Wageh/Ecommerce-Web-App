# 🛒 E-Commerce Web Application

## 📌 Overview

This project is a fully responsive **E-Commerce web application** developed as a graduation project during the DPEI Scholarship.
The platform allows users to browse products, view details, and simulate an online shopping experience.

---

## 🚀 Features

* 🛍️ Product listing and categories
* 🔍 Product details page
* 🛒 Shopping cart functionality
* ❤️ Add to favorites (if implemented)
* 📱 Fully responsive design (Mobile / Tablet / Desktop)
* ⚡ Fast and clean UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox / Grid)
* JavaScript (ES6)

---
